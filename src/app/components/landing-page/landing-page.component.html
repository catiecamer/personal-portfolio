Projects

UNIVERSITY OF WATERLOO PROJECT

We had a big challenge when approaching the Waterloo website: Organizing and presenting large amounts of information to various audiences. The project was completed in a three-week timeline and included two team members.

THAT TORONTO STUDIO

In a three-week design project for THAT Toronto Studio, I was tasked with suggesting UI and UX improvements for their current desktop and mobile websites based on user feedback. The focus was on enhancing the landing page and booking process, which were identified as crucial elements in improving the overall user experience.

CANADA COMPUTERS

The following is a UX design case study for the Canada Computers website completed by five students at General Assembly. With a timeline of two weeks, we directed our focus to the checkout process, which most users identified as pain points in early rounds of task analysis.

GMAIL

As part of a General Assembly UX Design project, I was given the task to research and improve a pre-installed android application with a two-week turnaround. The following is an analysis about the user friendliness of the Gmail application, more specifically when it comes to composing emails and interacting with ‘drafts’ on an android.
